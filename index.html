ELECTRONIC TABLA (MIT MEDIA LABS-DESIGN INNOVATION INDIA)

×

Design Innovation is an initiative by MIT Media Labs to encourage innovation. I was part of Sensors across scale track where I teamed up to build Electronics Table also know as TAAL. Unlike any other musical instruments a traditional tabla did not undergo any revolution. So we decided to add some modern technology to it. We interfaced capacitive touch sensors to the frame of tabla and those were interfaced over arduino to a sound processing software ABLETON over arduino midi interface. We implemented various filters on the sound coming from tabla so to add a new dimention to it.
