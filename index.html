ELECTRONIC TABLA (MIT MEDIA LABS-DESIGN INNOVATION INDIA)

Design Innovation is an initiative by MIT Media Labs to encourage innovation. I was part of Sensors across scale track where I teamed up to build Electronics Table also know as TAAL. Unlike any other musical instruments a traditional tabla did not undergo any revolution. So we decided to add some modern technology to it. We interfaced capacitive touch sensors to the frame of tabla and those were interfaced over arduino to a sound processing software ABLETON over arduino midi interface. We implemented various filters on the sound coming from tabla so to add a new dimention to it.

IoT LAB (Eduvance)

I was product lead for IoT lab offering by Eduvance. I worked on ESP8266 and HTML5 to create IoT lab devices. Also created REST APIs over GET requests so that it can be used in any IoT applications. To know more about these products follow the link IoTlab product page

LPC2148 Grader-GradeME (Eduvance)

GradeME is another offering by Eduvance which is a skill development and assesment platform. I made a module to test an individual's skill on ARM7 architecture using LPC2148. Whole module was written in python2.7 and using LPC2148 simulator. It evaluated skills of a student on concepts based on ARM7 using programming assignments given to him using checkpoints based approach. The module was implemented as a web worker and communicated to the platform using GradeMe framework. Follow the link to checkout GradeMe
